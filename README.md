# Python Library for Contest

> This repository is a useful python library for Contest.
>
> 유용한 파이썬 라이브러리 소스코드 저장소입니다.



## Content

> - **itertools**
>   - 순열, 조합 라이브러리
> - **heapq**
>   - 힙(Heap) 자료구조
>   - 우선순위 큐 구현
> - **bisect**
>   - 이진 탐색(Binary Search)
> - **collections**
>   - 덱(deque), 카운터(Counter)
> - **math**
>   - factorial, sqrt, 최대공약수(GCD), 삼각함수, 파이(pi) 등
